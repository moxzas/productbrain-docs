## Highlights

### The Planning Tree responds predictably

Expand, collapse, and focus behave the same way every time — switching between Overview and iterations, creating new nodes, committing drafts. No unexpected jumps, no collapsed branches hiding your work.

### Overview is visually distinct

Overview mode now has its own look — green-tinted background, solid connection lines. Iteration mode is purple with animated lines. You can tell which mode you're in at a glance.

### Goal visibility in one place

Goal visibility is now controlled entirely through the Focus panel. One mechanism, one place.

## Other improvements

- Approaches stay visible after committing new nodes
- Viewport includes recently committed work
- Inline drafts no longer jump the view
- Docs link points to the new documentation site
- Smoother view transitions

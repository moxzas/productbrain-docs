## Highlights

### Learn the model by exploring it

New users now walk through an interactive example before the wizard. A coffee shop planning tree demonstrates how goals, needs, approaches, and jobs connect — click each node, see how they relate. Teaches the framework through a real example, not a wall of text.

### Overview refined

Overview mode now uses a darker blue background — distinct from iteration mode but in the same colour family. Clearer visual signal without the clash.

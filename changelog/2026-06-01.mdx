## Highlights

### Tree view rebuilt from the ground up

The Planning Tree's expand, collapse, and focus behavior has been completely rewritten. The tree now responds predictably to every action — switching between Overview and iterations, creating new nodes, committing drafts. No more unexpected jumps or collapsed branches hiding your work.

### Overview is visually distinct

Overview mode (when no iteration is selected) now has its own look — a green-tinted background with solid connection lines. Iteration mode has a purple background with animated lines. You can tell at a glance which mode you're in without checking the dropdown.

### Simplified goal visibility

Goal visibility is now controlled entirely through the Focus panel. One mechanism, one place. Select which goals to show, and everything else follows.

## Other improvements

- Approaches stay visible after committing new nodes — no more disappearing branches
- Recently committed work is included in the viewport — you see what you just created
- Inline drafts stay in context — creating a need or approach doesn't jump the view
- Docs link in the menu now points to the new documentation site
- Smoother transitions when switching between Overview and iteration views
